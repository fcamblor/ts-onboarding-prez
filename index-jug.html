Welcome

Fin refacto Logger

  • Classe avec méthodes public static
  • Interfaces LogEventCallbackStdParam / LogEventCallbackUserParam
  • Type LogEvent qui ressemble a un enum
  • registeredObservers en private static
  • LogEventObserver qui est une Map
  • Externalisation de la classe Logger pour faciliter son export futur

Tricks

  • Petit focus sur les enums qui sont vraiment pourris en TS
  • let est mieux que var

NG-Controller en TS

  • Controller en tant que classe
  • Injection de dépendances via public static $inject
  • Possibilité de colocaliser le template avec le controller

Tricks

  • Fat arrows ne redéfinissent pas this, super, arguments et target.new

Export du logger

  • Export de WorkshopLogger : nécessite de clarifier l'usage de requirejs, pb gestion require/angular.module
  • Etapes :
    • Création d'une constante angular dans un fichier à part, objectif : le charger
    • Réorga de requirejs dans le index.html => changement du comportement de certaines libs (angular-ui-router par ex)
    • Suppression ng-app + déclaration modules pour angular / ui-router
    • Import de la constante dans le main
    • Migration de tous les controllers en TS + déclaration de chacun d'eux dans le app.ts
  • Remplacer le logger injecté via angular par l'appel static à la classe WorkshopLogger

Tricks

  • Live template "angularrequire"
  • Re-lancer la compil typescript dans Intellij pour tout bien prendre en compte
  • Le controller peut implémenter ng.IController
  • $http peut être défini via un IHttpService
  • Le component peut implémenter ng.IDirective
