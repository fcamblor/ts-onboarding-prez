Refacto classe Logger

  • Classe avec méthodes public static
  • registeredObservers en private static
  • Type LogEvent qui ressemble a un enum
  • Interfaces LogEventCallbackStdParam / LogEventCallbackUserParam
  • LogEventObserver qui est une Map
  • Externalisation de la classe Logger pour faciliter son export futur

Migration Typescript d'un controller

  • Controller en tant que classe
  • Injection de dépendances via public static $inject
  • Possibilité de colocaliser le template avec le controller
  • Possibilité de spécifier la structure des données manipulées

Tricks

  • Le controller peut implémenter ng.IController
